Atividade) Nesta atividade o foco é desenvolver um programa inspirado em uma venda de passagens de
ônibus. Imagine que um operador de um guichê precisa realizar vendas de passagens de um ônibus com 44 lugares. O programa deve oferecer as seguintes opções de menu:

# Opção:

(I) Inicializar. Inicializa o ônibus com todas as poltronas livres, prontas para serem
vendidas.

(V) Visualizar. Mostra o ônibus com suas poltronas dispostas à semelhança do
ônibus. Veja figura abaixo de exemplo de um site de venda de
pesagens.

(P) Consultar. Dado um número de poltrona, indica se a poltrona é janela ou corredor se de ela está ocupada ou vazia.

(E) Vender. Realiza a venda uma passagem ocupando a poltrona informada.

(C) Cancelar venda. Realiza o cancelamento de uma venda de passagens anterior

(R) Relatório. Relatório de ocupação do Ônibus (percentual de poltronas
vendidas), percentual de janela, percentual de corredor e o total
financeiro vendido. Considere que o valor da passagem é R$53,20.

(S) Sair. Encerra o programa

Comentários:

- Como esta é uma atividade para avaliar o crédito de modularização, crie
apropriadamente a função de menu e para cada opção de menu invoque a função
correspondente.
